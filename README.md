# Gmail AI: Strategic Email Categorization

Efficiently organize your Gmail using a hybrid of Gemini AI and local Machine Learning.

## 🚀 Quick Start
```bash
# 1. Setup
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python scripts/setup_credentials.py

# 2. Daily Use
python main.py --service  # Runs the 24h classification service
```
